# HTML Projects

This repository contains HTML practice programs, exercises, and mini projects created while learning web development.

## Topics Covered

- HTML Basics
- Document Structure
- Headings and Paragraphs
- Links and Images
- Lists
- Tables
- Forms
- Semantic HTML
- Audio and Video
- HTML5 Elements

## How to Use

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Open the project folder.

3. Open any `.html` file in your web browser.
